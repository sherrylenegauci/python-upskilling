# python-upskilling

A starter Python repository for upskilling and sample scripts.

## Project structure

- `src/python_upskilling/`: package source code
- `tests/`: unit tests
- `pyproject.toml`: build configuration

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e .
```

## Run

```bash
python -m python_upskilling.main
```

## Test

```bash
python -m pytest
```
